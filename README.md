### ShadowWarrior2ArchiveTool
A simple tool for pack and unpack .bin archives from Shadow Warrior2 game. Based on this [script](http://aluigi.org/bms/shadow_warrior_2.bms) for QuickBMS

#### Usage

Unpack

```
ShadowWarrior2ArchiveTool.exe unpack [path_to_bin] [out_folder]
```

Pack
```
ShadowWarrior2ArchiveTool.exe pack [out_folder] [path_to_bin]
```
Note: for pack you must write root path whete data folder starts
